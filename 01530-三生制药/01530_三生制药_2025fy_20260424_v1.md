## 一、业绩快照

2025年是三生制药历史上最具里程碑意义的一年。公司全年总营收约176.96亿元，同比大增94.3%；归母净利润约84.82亿元，同比飙升305.8%，基本每股收益从0.86元人民币跳升至3.51元人民币。

爆发式增长的核心驱动是单一事件：公司于2025年5月与辉瑞（Pfizer）签署PD-1/VEGF双特异性抗体 SSGJ-707 的全球许可协议，由此一次性确认授权许可收入约94.26亿元人民币（另含CDMO相关药物原料销售约9506万元）。这笔收入是报告期内唯一来自美国市场的客户，占总营收的53.4%。

剥离BD授权收入后，公司核心生物药销售约80.06亿元，同比下滑约10.3%，主因集采降价及医保政策压缩了特比澳（TPIAO）、益比奥（EPIAO）等传统产品的收入空间。核心业务承压是本期不能回避的负面信号。

毛利率92.4%，相较2024年的86.0%提升6.4个百分点，主要是因授权收入（几乎零成本）大幅拉高了整体毛利水平；若仅看生物药业务，毛利结构并无实质改善。

### 2.2 毛利率与费用率联动

- **生物药核心业务毛利率**：依据成本结构推算，传统生物药毛利率约85-87%，处于行业较高水平，但销售费用率仍约为销售收入的45%+，说明产品仍需大力推广，护城河有一定依赖推广投入维护的特征。
- 销售及分销费用率：以总营收计为20.5%（2024年：36.8%），大幅下降，但如以剔除BD收入的核心销售收入计，实际费用率仍接近45%，与历史持平。
- 研发费用率：以总营收计8.6%，以核心业务收入计约19%，研发投入占核心收入比例合理甚至偏高，符合创新药企特征。
- 财务费用从1.91亿元降至0.76亿元，降幅达60%，熊猫债偿还带来实质性财务减负。

### 2.3 现金质量初判

经营现金流96.69亿元，净利润90.96亿元（含少数股东），经营现金流覆盖净利润比率约1.06，整体合格。但应注意授权收入的现金回收节奏是关键变量（辉瑞首付款14亿美元已实际到账，印证了现金流真实性）。

## 四、现金流量表深挖

年报公告中直接列出了关键现金流数据，现综合整理：

| 项目 | 2025年（亿元） | 2024年（亿元） |
|------|-------------|-------------|

| 期末现金及等价物 | 121.77 | 21.43 |
| 现金类总金融资源 | 204.0 | 约 41.7 |

**① 造血能力：** 经营现金流96.69亿元 vs 净利润90.96亿元，经营现金流/净利润比约1.06，基本健康。更精准看：核心利润获现率（经营现金流 ÷（核心利润+其他收益））≈ 96.69 / （约13亿核心业务利润+约107亿BD相关利润混合体）结构较复杂，整体看现金回款速度快，BD首付款已实际入账并体现在现金流中。

**② 用血方向：** 公司未在简报中单独披露资本开支明细，但年报指出未来三年预算资本支出12-15亿元。固定资产规模几乎无增长，说明2025年资本支出较低。配股和辉瑞认购净得资金共约35-40亿港元，绝大部分将用于研发（80%）和一般营运（20%）。

**③ 输血状况：** 公司在2025年12月配股筹资约30.87亿港元（净额），辉瑞认购7.85亿港元；同时偿还了12亿元熊猫债及部分银行借款。筹资呈现"适度外部融资+偿还旧债"结构，经营造血良好，不依赖持续输血。

**④ 三阶段判断：** 经营+（充裕），投资-（适度），筹资混合（融资+还债结构）。总体属于成熟成长型企业结构，2025年因BD事件呈现异常正的经营现金流，为后续研发投入积累了强厚资本储备。

3年/5年有机增长率约5-8%，处于中等偏下水平。集采降价是中长期持续压力。

### 增长驱动分析

**传统核心产品（约占2025年生物药收入主体）：**
- **特比澳（TPIAO）**：全球唯一商业化rhTPO，2025年市占率60.4%（按销售额），全年销售约50.62亿元（+20.4%），但二级市场中长期受仿制药潜在入市和医保续约压价威胁。新增CLDT适应症（慢性肝病相关血小板减少症）于2025年12月获批，有增量空间。
- **rhEPO产品（益比奥/赛博尔）**：市场份额39.6%，受集采降价持续压缩单价，量价博弈下整体微增。
- **蔓迪（Mandi）**：米诺地尔市场领导者，OTC属性强，受监管影响小，成长性好，拟分拆上市（Mandi Inc.已申请HKEx上市）。

**创新驱动未来（2026-2028关键催化剂）：**
- SSGJ-707（已授权辉瑞，辉瑞主导全球III期）：VEGF/PD-1双抗，三生保留中国权利和全球供货权，里程碑付款+分成是未来弹性来源。
- 608（Yisaituo，抗IL-17A单抗）：2026年2月获批，适应症为中重度斑块状银屑病，即将商业化。
- 613（抗IL-1β单抗）：急性痛风性关节炎NDA已受理。
- SSS06（长效EPO）：NDA已获批，透析贫血适应症。
- 611（抗IL-4Rα）：AD（特应性皮炎）NDA受理中，对标度普利尤单抗（Dupixent），市场空间大。

**护城河识别：**

| 类型 | 评估 |
|------|------|
| **无形资产（专利/许可）** | ✅ 较强。TPIAO是全球唯一商业化rhTPO，专利保护。707与辉瑞合作印证国际顶级制药公司对其平台认可。 |
| **转换成本** | 中等。医院/医生用药习惯形成一定粘性，但生物类似药上市后切换成本降低。 |
| **规模效应** | 中等。国内销售网络覆盖3600+三甲医院，渠道壁垒较高。 |
| **网络效应** | 无 |

综合判断：**窄至中等宽度护城河**。TPIAO的产品壁垒是核心，蔓迪的品牌效应在细分市场较强，但传统生物药（rhEPO、Yisaipu）面临生物类似药竞争，护城河正在被侵蚀。创新管线（607、608、611等）若能规模上市，将构建新的创新药护城河，当前处于转型窗口期。

## 八、投资视角总结

三生制药2025年报是一份"结构性超预期"的业绩，BD授权收入带来了营收和利润的历史峰值，但其可持续性有限。公司核心优势在于：**唯一商业化rhTPO产品（特比澳）构成无形资产型护城河**，长期市占率超60%；以及**通过707授权辉瑞获得了国际级别背书和充裕的研发资金**，为后续创新管线推进奠定强大的资本和信誉基础。

主要风险有二：其一，**传统生物药业务受集采降价和医保政策持续承压，2025年核心业务同比下滑10%，2026年若集采继续深化，基础盘仍将承压**；其二，**2026年利润将大幅回落至约15-20亿元正常化水平（较2025年84.8亿元下降约75-80%），业绩落差可能引发市场情绪波动**。此外，美元外汇敞口（约99.5亿元等值）在人民币走强环境下存在汇兑损失风险。

估值结论：**合理区间 HKD 20-28**，当前24.82港元处于合理区间中段，价格吸引力中等。若股价回落至19-22港元区间，则具备较好的安全边际，可作为阶段性买点参考。这份财报整体定性为**"一次性超预期，结构性需观察"**。
